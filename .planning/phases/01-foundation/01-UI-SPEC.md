## Spacing Scale

Declared values (multiples of 4). Extracted from existing Hero.tsx and NavBar.tsx:

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding |
| sm | 8px | NavBar pill inner vertical padding (`py-2.5` ≈ 10px, nearest token) |
| md | 16px | Default element spacing, horizontal gaps |
| lg | 24px | Section padding, gap between nav links (`gap-10` = 40px at nav level) |
| xl | 32px | Layout gaps between major sections |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

Exceptions:
- Hero section bottom padding: 48px (`pb-12`)
- Hero section top padding: 96px (`pt-24`) — reserves space for fixed NavBar
- NavBar horizontal pill padding: 24px (`px-6`)
- NavBar link horizontal padding (Contact CTA): 20px (`px-5`)
- Hero bio bottom offset: `bottom-16` (64px from viewport bottom)
- Hero bio right offset: `right-10` (40px from viewport right)
- Touch target minimum: 44px (Contact CTA pill meets this via combined padding + font height)

## Typography

All type is set in Geist. Fluid sizes use `clamp()` — declared values are the
computed range, not a fixed px value.

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Display (split text) | clamp(72px, 10vw, 160px) | 900 (font-black) | 1 (leading-none) | Hero "Data" / "Stories" background text |
| Heading (name label) | 12px (text-xs) | 600 (font-semibold) | default | "Nikki Jiang" eyebrow above photo |
| Body (bio) | clamp(12px, 1vw, 15px) | 400 (regular) | 1.625 (leading-relaxed) | Hero bio bottom-right |
| Label (nav links) | 14px (text-sm) | 400 regular / 500 medium (Contact CTA only) | default | NavBar link text |

Letter spacing:
- Name label only: `tracking-widest` + `uppercase` — signals identity label, not body copy
- All other text: default tracking

Weight rule: exactly 2 weights in use — regular (400) and semibold/black for emphasis.
The display split text uses font-black (900) as a decorative typographic element, not a
content weight. Content weights are regular (400) and medium (500) for the Contact CTA.

## Color

Extracted from existing codebase. Warm and personal — not dark editorial.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#ffffff` (white) | Page background, NavBar pill background, photo card shadow base |
| Secondary (30%) | `#f5f5f5` neutral-100 / `#e5e5e5` neutral-200 | NavBar pill border, future card surfaces, sidebar areas |
| Accent (10%) | `#171717` neutral-900 | Hero split text fill, Contact CTA button background, body text |
| Destructive | not applicable | No destructive actions in Phase 1 |

Accent reserved for:
1. Hero display split text ("Data" / "Stories") — decorative, behind photo
2. Contact CTA pill button background in NavBar
3. Primary text color (`text-neutral-900`) on body copy where emphasis is needed

Mid-tone in use:
- `#737373` neutral-500 — bio copy, name label, nav link default state
- `#404040` neutral-600 — nav link default hover target color
- `#404040` → `#171717` on hover (nav links) via `hover:text-neutral-900`
- `#404040` neutral-700 — Contact CTA hover state (`hover:bg-neutral-700`)

Color rule: warm white + neutral grays only. No blue, no teal, no brand color.
Warmth comes from Geist typeface + rounded corners, not hue.

## Phase 1 Visual Constraints (Summary)

1. **No new visual language.** This phase fixes text and wiring. The design system is
   the one already in place in the codebase.
2. **Preserve all class names** on NavBar and Hero exactly — only update the data/content
   strings inside them.
3. **Stub pages are invisible scaffolding.** Minimal markup; no design investment.
   They exist to stop 404s and will be replaced in later phases.
4. **Geist is the only font.** Do not introduce any additional typeface.
5. **White + neutrals only.** Do not introduce any hue (blue links, colored tags,
   brand color) in this phase.
6. **clamp() preserves fluid type.** Do not convert fluid sizes to fixed px.
7. **No animation.** CSS transitions only, already present on NavBar hover states.
